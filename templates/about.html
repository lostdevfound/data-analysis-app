The project was created in my spare time purely for learning and experimenting with with mathematical concepts of Fourier Series. The algorithm implementation can be found on Github. The webapp source code can be also found on Github.
Note: The webapp is hosted on the limited Heroku account, therefore the performance is quite slow.

Computing Stack:
  • Python 3.6
  • NumPy - scientific computing module
  • SymPy - symbolic mathematics
  • Matplotlib - 2d plotting library
Web Stack:
  • Python 3.6
  • Flask - Python micro web framework
  • Jquery - JavaScript front-end framework
  • Bokeh - Python interactive visualization library for web browsers
  • Bootstrap CSS - front-end framework
References:
  • Fourier Analysis
  • Fourier transform
  • Convolution
